About This Website

A static prototype gallery built for CSC 340 Assignment 2.

Purpose

This site is a static web app prototype designed to practice building a multi-page website with consistent navigation, organized content, and a clean visual layout. There is no backend—everything is built with HTML, Bootstrap styling, and a small amount of client-side JavaScript for the search bar on the home page.

Why Naruto Shippuden?

I chose Naruto Shippuden because it has iconic characters with clearly defined roles that fit perfectly into a gallery format. The cast includes different archetypes—an energetic main hero, a talented rival with a complex story, a powerful medical ninja, a veteran mentor, and a legendary teacher. That makes it easy to present short summaries on the home page and expand into detailed profiles on the details page.

Design Choices

For the visual design, I used Bootstrap’s “Album” style layout because it naturally creates a neat, responsive grid of cards that looks professional. I also used a dark theme for strong contrast and readability. Each card includes an image, a short description, and a button linking to the details page. The navigation bar stays consistent across all pages to make the site easy to use.

How to Use the Site

  • Home: Browse the gallery and use the search bar to filter characters.
  • Details: View extended profiles with more information about each character.
  • Add a Character: Fill out a prototype form to “add” a character (not functional).

Credits / Notes
